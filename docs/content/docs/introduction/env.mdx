---
title: Environment Variables
description: Complete reference of every environment variable in Mannaiah, grouped by module.
---

All variables are read from the process environment. Copy `.env.example` to `.env` and fill the
required values before starting the service.

Variables marked **required** have no default and will cause a startup validation error if absent.

---

## Core

Controls the HTTP server host, port, and runtime environment label.

| Variable | Default | Required | Description |
|----------|---------|----------|-------------|
| `CORE_HOST` | `0.0.0.0` | No | Bind address for the HTTP server |
| `CORE_PORT` | `8080` | No | Port the HTTP server listens on |
| `CORE_ENVIRONMENT` | `development` | No | Environment label (`development`, `production`, etc.) |
