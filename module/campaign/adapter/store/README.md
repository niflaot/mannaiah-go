# Campaign Store Adapter

GORM repository for campaign definitions and send state counters.

## Key methods / endpoints / events
- Methods:
  - `NewRepository(db)`
  - `(*Repository).Create(...)`
  - `(*Repository).GetByID(...)`
  - `(*Repository).List(...)`
  - `(*Repository).Update(...)`
  - `(*Repository).Delete(...)`
- Endpoints: none.
- Events: none.
