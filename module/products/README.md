# Products Module

Product catalog module with multi-realm datasheets, gallery composition, and CRUD endpoints.

## Key methods / endpoints / events
- `POST /products`
- `GET /products`
- `GET /products/:id`
- `PATCH /products/:id`
- `DELETE /products/:id`
- `POST /variations`
- `GET /variations`
- `GET /variations/:id`
- `PATCH /variations/:id`
- `DELETE /variations/:id`
- No integration events are emitted yet.
