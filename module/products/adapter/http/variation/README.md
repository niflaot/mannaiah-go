# products/adapter/http/variation

Fiber HTTP handlers for variation CRUD endpoints.

## Key methods / endpoints / events
- Methods: `NewHandler(service, authorizers...)`, `(*Handler).SetAuthorizer(authorizer)`, `(*Handler).RegisterRoutes(router)`
- Endpoints: `POST /variations`, `GET /variations`, `GET /variations/:id`, `PATCH /variations/:id`, `DELETE /variations/:id`
- Events: none.
