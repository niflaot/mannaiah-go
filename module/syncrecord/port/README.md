# Sync Record Port Package

Defines repository and recorder abstractions for application services and external module injection.

## Key methods / endpoints / events
- Methods:
  - `Repository.CreateRun(...)`
  - `Repository.CompleteRun(...)`
  - `Repository.AddRunErrors(...)`
  - `Recorder.StartRun(...)`
  - `Recorder.CompleteRun(...)`
  - `Recorder.FailRun(...)`
- Endpoints: none.
- Events: none.
